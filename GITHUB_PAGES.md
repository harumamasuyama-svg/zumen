# GitHubだけでZUMENを公開テストする手順

1. GitHubで新しいリポジトリを作成します。
2. このフォルダの中身をすべてアップロードします。
3. GitHubのリポジトリ画面で Settings を開きます。
4. 左メニューの Pages を開きます。
5. Build and deployment の Source を GitHub Actions にします。
6. Actions タブを開き、Deploy ZUMEN to GitHub Pages が完了するのを待ちます。
7. 完了後、Settings > Pages に表示されるURLでZUMENを試せます。

更新したいときは、同じリポジトリにファイルを追加・更新すると自動で再公開されます。
